# Music Recommendation System

## Project Overview

This project is a Music Recommendation System developed using Python. It recommends songs to users based on patterns and similarity in music data. The system helps users discover new music based on their interests.

## Features

* Recommends songs based on similarity
* Simple user interface using HTML and CSS
* Database support using SQLite
* Efficient data processing using Python

## Technologies Used

* Python
* HTML
* CSS
* SQLite
* Machine Learning Concepts

## How to Run the Project

1. Clone the repository
2. Install required Python libraries
3. Run the main file

```
python main.py
```
